# Technology Stack

## Backend

- **Language:** Node.js
- **Framework:** Express.js
- **Real-time Communication:** Socket.IO
- **Authentication & Database:** Firebase (Authentication and Firestore)
- **Containerization:** Docker

## Frontend

- **Languages:** HTML, CSS, JavaScript
- **Libraries:** Firebase (Authentication Client SDK), Socket.IO Client SDK
- **Architecture:** Modular JavaScript with a focus on separation of concerns (state, UI, networking, etc.)

## Development & Operations

- **Dependency Management:** npm
- **Environment Variables:** dotenv
- **Security Middleware:** Helmet, csurf, cors
- **Process Management:** pm2 (implied for production in docker-compose)
